# Contents

Program allows you to create a .txt file and input lines of instructions.
An example of a full instruction is:

1 LET JON 1
2 LET BRENDAN 3
3 ADD JON BRENDAN
4 PRINTALL
5 END
6 .

Which would print out:

JON 4
BRENDAN 3

To run the program, type make in the compiler and run ./interpreter
